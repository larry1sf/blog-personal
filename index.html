Tener cuenta para seleccionar las actividades que ya se allan realizado en el dia.

Actividad Hora de registro Hora de finalizacion Checkeo

instrucciones

Selecciona la actividad junto con la hora de inicio, finalizacion de registro y si se realizo en el dia para almacenarlo en la base de datos web.

Pulsar el boton "Añadir registro" para añadir registro en la base de datos web.

Pulsar el boton "Borrar todo" para eliminar toda la informacion de la tabla y reiniciar sus datos. Tambien existe la opcion de eliminar un solo registro dandole a la ✖

El boton de "Enviar de datos" almacena los datos registrados en la tabla para poder observarlos en la pagina que se creara con la fecha estimada.
